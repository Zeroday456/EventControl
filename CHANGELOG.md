# EventControl Changelog

## [1.0.0] - 2026-06-30

### Добавлено
- Создание событий с названием, датой и описанием
- Валидация входных данных при создании события
- Получение информации о событии (название, дата, описание)
- Проверка валидности события (isValid)
- Иммутабельность класса Event (поля final)
- Модульные тесты для класса Event (JUnit 5)

### Исправлено
- Все ошибки Checkstyle (56 нарушений стандартов кодирования)
- Конфликты в конструкторе Event (устранено скрытие полей)
- Исправлены тесты для иммутабельного класса

### Технические детали
- Использован Java 21
- Сборка через Maven
- CI/CD настроен через GitHub Actions
